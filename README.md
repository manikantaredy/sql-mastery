## 👨‍💻 About This Repository

This repository documents my complete SQL learning journey, structured into **4 phases** with **100 practice questions each** (50 Easy + 30 Medium + 20 Hard).

Every query is written and tested on **MySQL** using a custom `employees` and `products` dataset — no theory-only learning here. Every answer runs and produces real output.
